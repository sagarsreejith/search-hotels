hotel-search

GitHub Search

The main goal of the project is, to create a sample project in React. Based on user input selection, fetch User or Repository data from GitHub API and store the result in the redux-store. If the API response is cached show the result from indexedDB.

Libraries used in this project
  • React
  • Redux
  • Redux Thunk
  • Redux Router
  • Redux Persist
  • Lodash
  • Sass

Usage

User can able to search users and users repository based on the user input.

This application has two input fields.

  1. Text Input[textbox]

    User can enter a string. The string could be User's Name or User Repository Name.

  2. Select Input Box

    User can select the search type from a select box. The option could be Users or Repository.

Based on these two user inputs app will perform fetch data from GitHub API. After getting successfull response, the data stored in the redux-store. The app will pull the data from redux-store if the current serch text is present in the redux-store. For better experience add a debounce time for the user input text box.

store.ts

Redux persist takes app Redux state object and save it to Persistence storage. For z is wrap your app’s root reducers and pass it to the persistStore function it ensures your redux state is stored to persisted storage whenever it changes. Redux Thunk middleware allows us to write action creators that return a function instead of an action. The thunk can be used to delay the dispatch of an action, or to dispatch only if a certain condition is met. The inner function receives the store methods dispatch and getState as parameters. persistConfig sorage we are using localforage.


import {
 createStore,
 applyMiddleware,
} from 'redux';
import { composeWithDevTools } from 'redux-devtools-extension';
import thunk from 'redux-thunk';
import {
 persistStore,
 persistReducer,
} from 'redux-persist';
import localforage from 'localforage';
import { rootReducer } from './reducers/RootReducer';
const persistConfig = {
 key: 'root',
 storage: localforage,
};
const persistedReducer = persistReducer(
 persistConfig,
 rootReducer
);
export const store = createStore(
 persistedReducer,
 composeWithDevTools(applyMiddleware(thunk))
);
export const persistor = persistStore(store);

TherootAction action creator will handlle the api call and read from localstore operations and the reullt will dispatch to rootReducer with appropriate action.type and action.payload.

Finally the reducer will change the state based on type and payload values.

   * @param {string} searchText - first inpus as {string}

Based on the return value of hasData function, the triggerSearch function will call an action creator (getDataFromLocalStorage or getDataFromApi). 

....

  const triggerSearch = _.debounce(
    (event: KeyboardEvent<HTMLInputElement>) => {
      const element = event.target as HTMLInputElement;
      if (element?.value?.length > 2) {
        !isDataStored(element?.value)
          ? dispatch(
              getDataFromApi(
                element.value,
                state.searchType
              )
            )
          : dispatch(
              getDataFromLocalStorage(
                element.value,
                state.searchType
              )
            );
      } else {
        dispatch({
          type:
            ActionsEnum.CLEAR_REPO_OR_USER_LIST,
          payload: {
            SearchText: element.value,
            searchType: state.searchType,
          },
        });
      }
    },
    900

getDataFromLocalStorage function will dispatch an action to the rootReducer. 


....

export const getDataFromLocalStorage = (
  searchText: string,
  searchType: string
) => async (dispatch: Dispatch<ActionType>) => {
  const payload = {
    searchText,
    searchType,
  };
  dispatch({
    type: ActionsEnum.SAVE_AND_SHOW_LOADER,
    payload,
  });
  dispatch({
    type: ActionsEnum.LOAD_FROM_HISTORY,
    payload,
  });
};

The reducer makes changes to the application state based on the action type and update the store.

...

switch (action.type) {
    case ActionsEnum.SAVE_AND_SHOW_LOADER: {
      return {
        ...state,
        isLoading: true,
        currentSearchText:
          action.payload.searchText,
        searchType: action.payload.searchType,
        error: {
          hasError: false,
          message: '',
        },
      };
    }

    case ActionsEnum.LOAD_FROM_HISTORY: {
      return {
        ...state,
        isLoading: false,
        currentSearchText:
          action.payload.searchText,
        searchType: action.payload.searchType,
        usersToDisplay:
          state.locallyStoredUsers.find(
            (element) =>
              element.searchText.toLowerCase() ==
              action.payload.searchText.toLowerCase()
          )?.users || [],
        repositoriesToDisplay:
          state.locallyStoredRepositories.find(
            (element) =>
              element.searchText.toLowerCase() ==
              action.payload.searchText.toLowerCase()
          )?.users || [],
        error: {
          hasError: false,
          message: '',
        },
      };
    }
}
